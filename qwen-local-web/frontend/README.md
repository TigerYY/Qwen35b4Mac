# Qwen AI Terminal 前端工作区 (Frontend)

这是为 Qwen 3.5 (35B-A3B) MLX 本地引擎打造的高颜值、支持多模态（图片上传）的专属暗色极光风格交互界面。

基于 **React** 和极速构建工具 **Vite** 开发。它能自动嗅探本地环境下的后端自动探寻端口机制。

---

## 🚀 如何启动前端页面

只需两步命令，即可唤起炫酷的聊天终端：

1. **进入前端专用目录**:
   打开一个新的终端窗口，确保您处于本项目的前端文件夹下：
   
   ```bash
   cd /Users/yangyang/Documents/YYLLM/qwen-local-web/frontend
   ```

2. **启动本地开发服务器**:
   
   ```bash
   npm run dev
   ```

3. **访问页面**:
   服务启动后（通常仅需毫秒级），终端会吐出一个本地链接（如 `http://localhost:5175`）。按住 `Cmd` 键并点击该链接，即可打开专属 AI Web 界面。

> **💡 提示**: 在启动前端前，请务必确保位于根目录的 `./start_backend.sh` 已经处于常驻运行状态，这样当您在网页里发出第一条消息时，才会有大模型处理您的请求！
